# User Stories

| En tant que | Je veux | Afin de |
| ----------  | ------- | ------- |
| Utilisateur | visualiser le board complet | voir les différentes listes et cartes qui le composent |
| Utilisateur | créer une liste | lister les différentes étapes d'avancement du projet |
| Utilisateur | modifier une liste | changer son titre |
| Utilisateur | supprimer une liste | ... |
| Utilisateur | déplacer une liste | réordonner le board |
| Utilisateur | créer une carte | ajouter une nouvelle carte dans une liste |
| Utilisateur | modifier une carte | mettre à jour son titre |
| Utilisateur | supprimer une carte | ... |
| Utilisateur | déplacer une carte | réordonner une liste ou changer la carte de liste |
| Utilisateur | créer un label | ... |
| Utilisateur | modifier un label | changer son nom et sa couleur |
| Utilisateur | supprimer un label | ... |
| Utilisateur | associer un label à une carte | catégoriser la carte |
| Utilisateur | dissocier un label d'une carte | ... |
